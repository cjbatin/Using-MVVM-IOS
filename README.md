# Weather Forecast

An example project that accompanies my tutorial on decoding JSON using Codable in
Swift 4. **Don't forget to link back to the tutorial.**

## Getting Started

Clone or download the repo onto your machine and open it in Xcode.

### Prerequisites

* Xcode 9.3+
* Swift 4.1+
